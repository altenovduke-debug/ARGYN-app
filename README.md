# Argyns — Family Registry

![Argyns App Icon](assets/images/app_icon_opaque.png)

**Argyns** — это современный цифровой реестр 12 родов Аргын. Приложение создано для объединения потомков, сохранения истории и укрепления связи поколений. Наша цель — не разделять, а объединять.

## 🌟 Ключевые функции
*   📊 **Единая база данных**: Актуальная статистика представителей племени.
*   🔢 **Счетчик рода**: Визуальный контроль численности вашего рода.
*   🔔 **Уведомления**: Важные новости и съезды сородичей.
*   📍 **Геолокация**: Регистрация с указанием города.
